css style sheet

css 작성 할 때는 외부스타일과 내부스타일 방식이 있다. 내부 스타일은 <head> 태그 내에 <style>>태그로 작성하고, 외부스타일은 별도의 css 파일을 생성하여 <link>태그로 연결하는 방법이다.

  • 아이디는 css에서 #으로 표시
  • 태그는 css에서 꺽쇠 없이 태그명만 표시
  • 디자인하는 대상이 정해졌을 때
  • 그 대상에 가장 가까운 태그의 태그명 또는 아이디, 클래스를 확인하고 선택자 작성
자식 em span 안 두 번째 자손 em 위치

작성 방법

주의사항

속성 값 끝날 때마다 세미콜론(;)작성

  • 영어 소문자로 클래스, 아이디 작성
  • 공백 특수문자 불가

CSS 속성색상

CSS에서 작성할 수 있는 색상 값의 종류

color, background-color

  • 헥사코드
  • #으로 시작하는 6자리 RGB코드 #ffff00
  • 6자리 모두 같은 글자 반복 시 3자리로 줄여쓰기 가능
RGBA
RGB와 alpha(A) 투명도를 함께 쓸 수 있는 코드
Asms 0~1까지의 값만 작성, 1은 100%와 같음
rgba(255,0,0,0.2)는 빨간색, 투명도 20%를 뜻함

선택자

태그, 아이디, 클래스 선택자

태그와 아이디 또는 클래스가 함께 있을 경우는 아이디, 클래스를 선택자에 먼저 작성

자식, 자손, 선택자

특정 부모 요소에 자식, 자손 태그가 중복되었을 경우만 >표시를 사용하여 자식 구분

형제 선택 자

인접한 형제는 + 기호 사용

모든 형제는 ~ 기호 사용
